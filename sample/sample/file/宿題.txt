7/23 宿題
	・オーバロードとオーバーライドが説明できる
		●オーバロード
			同じ名前のメゾット、仮引数は違う。
		●オーバーライド
			継承するとき、同名のフィールドやメゾットを定義。
		
	・継承の際のコンストラクタについて注意することを説明できる
		●仮引数をもつ場合、コンストラクタ内でsuper(引数)とする。
		　しない場合自動でsuper()となりエラー。
		
	・staticフィールドとインスタンスフィールドがオーバーライドされた時の違い
		●結果はソースTest2。理由はわからなかった。
		
	・以下の場合にどうなるかがわかる
		- 親の型に子のオブジェクト
		  Parent p = new Child();
		- 親の方の入れた子のオブジェクトを子の型に戻す方法
		　Child c = p;
		- 子の型に親のオブジェクトを入れるとどうなるか
		  Child c = (Child)new Parent;
		  
		  ●結果はソースTest。理由はわからなかった。
		  
	・キャストについて調べる
		●キャストとはプログラマが意識的に行う変換処理。
		　(変換したい型名)変数
		　サイズ大→サイズ小、はできる。
		　サイズ小→サイズ大、は桁があふれる。
